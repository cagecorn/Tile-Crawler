# 센티넬 AI

## 개요
센티넬은 플레이어를 호위하는 방어형 용병이다. 이동과 행동은 행동 트리 기반의 간단한 우선순위 규칙을 따른다.

## 행동 원칙
1. **근접 수비**: 기본적으로 플레이어에서 2~4타일 거리를 유지하며 주변을 맴돈다.
2. **추적 복귀**: 플레이어와 6타일 이상 벌어질 경우 즉시 플레이어에게 귀환하는 경로를 찾는다.
3. **시야 기반 교전**: 자신의 시야 안에 적을 발견하면 최단 경로로 접근하며, 마지막 발걸음으로 적과 겹치는 타일로 진입해 전투를 유도한다.
4. **돌파 차단**: 전투 중에는 플레이어보다 앞쪽(적과의 사이) 위치를 우선 목표 지점으로 삼아, 플레이어를 가로지르는 적의 길목을 차단한다.
5. **겹침 방지**: 다른 파티원이나 플레이어가 위치한 타일은 이동 경로에서 제외한다. 이미 점유된 위치는 우회하거나 다음 후보 지점으로 이동한다.
6. **전투 후 정렬**: 적을 처치하거나 더 이상 위협이 사라지면 플레이어 주변의 가장 가까운 빈 타일(2~4타일 권역)을 재정렬 목표로 삼는다.

## 행동 트리 스케치
- **Selector**
  - **공격 우선 순위**
    - 조건: 시야 내 적이 존재
    - 행동: 최근접 적까지 경로 계산 → 이동력만큼 전진 → 인접/동일 타일이면 전투
  - **거리 유지**
    - 조건: 플레이어와의 거리가 6타일 초과
    - 행동: 플레이어 위치로 복귀 경로 계산 → 이동력만큼 전진
  - **호위 순찰**
    - 조건: 기본 상태
    - 행동: 플레이어 주변 2~4타일 중 비어 있는 타일을 선택 → 경로 이동

## 파라미터 제안
- 이동력: 3칸/턴
- 시야: 9타일
- 우선 방어 목표: 플레이어와 적 사이의 최소 거리 타일

이 설계는 `designing_intelligence_behavior_tree_guide.md`의 우선순위 셀렉터 구조를 따르며, 트리의 각 노드는 독립적인 행동 모듈로 유지된다.
